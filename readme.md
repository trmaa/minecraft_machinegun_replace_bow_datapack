## Install

### Linux 

First, `git clone https://github.com/trmaa/minecraft_revolver_mod.git && cd minecraft_revolver_mod/`.
Then do `cp -r ./* ~/.minecraft/resourcepacks/revolver/` and `cp -r ./* ~/.minecraft/saves/<your_world>/datapacks/revolver/`.
